damerauDistanceLimited

language_table

制限付きDamerau-Levenshtein距離を計算する

2つの文字列間の編集距離を計算します。挿入、削除、置換、隣接する2文字の入れ替えをカウントします。 指定された最大距離を超えた場合は早期に計算を打ち切り、効率化を図ります。

Return Value

Type: size_t
