language_table

進捗状況の追跡と表示を行う構造体

長時間かかる処理の進捗状況を追跡し、コンソールに表示するための機能を提供します。 残り時間の推定や完了のレポート機能も含みます。

struct ProgressTracker {
size_t total;
size_t current;
size_t lastPercent;
StopWatch sw;
void initialize(size_t total);
void increment();
void finish();
}
